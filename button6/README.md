# Button Exercise 6 

### Objective

* 3 lights are controlled by 3 buttons
* the button that controls each light is random
* press the buttons in the correct sequence to win the game


### New Tools

* 1x 10k ohm resistor
* 1x 220 ohm resistor
* 1x button
* 1x led
* 3x jumper wires
